# nichcm.github.io — Portfólio Pessoal

Portfólio pessoal de **Nicolas Henrique Corrêa Martins**, desenvolvedor Full Stack.

🔗 **[nichcm.github.io](https://nichcm.github.io/)**

## Tech Stack

- **React 18** + Vite
- **Material UI v5** (dark theme)
- **react-markdown** + remark-gfm (renderização de README)
- **GitHub API** para listagem de repositórios públicos

## Seções

- **Sobre** — apresentação, foto e links sociais
- **Experiência Profissional** — timeline completo (Intelia, CodeByte, CEKSoftware e anteriores)
- **Formação Acadêmica** — FIAP, Universidade Positivo, SENAI + certificações
- **Competências Técnicas** — barras de progresso por categoria
- **Projetos** — repositórios públicos do GitHub com README expansível
- **Contato** — WhatsApp, e-mail, LinkedIn, GitHub

## Desenvolvimento local

```bash
npm install
npm run dev
```

Acesse em `http://localhost:5173`

## Deploy

O deploy é feito automaticamente via **GitHub Actions** a cada push na branch `main`.

> Certifique-se de que em **Settings → Pages** a source está configurada como **"GitHub Actions"**.
